# Random Picker App 🎯

A simple and intuitive web application designed to help you make decisions quickly by picking a random item from a list.

## ✨ Features
* **Easy to Use:** Just enter your options and click pick.
* **Fast & Lightweight:** Built with performance in mind.
* **Clean UI:** Simple interface for a better user experience.

## 🚀 Technologies Used
* **HTML5 & CSS3** - For the structure and styling.
* **JavaScript** - For the random logic.
